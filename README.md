# My own opengl engine

This is a project I wrote during my high school. It's very old and the code not optimal. I am refactoring and improving it right now.

![alt text](./engine.png)

## Features

- camera
- light
- objects
- obj loading
- terrain
- skybox
- post processing
- imgui
- shadows
